# 时光杂货铺 (Time Shop Game)

一个温馨治愈的模拟经营+叙事解谜游戏。

## 游戏概念

经营一家神奇的杂货铺，顾客来自不同的时代（过去或未来）。每个顾客都带着自己的故事和需求，你需要找到他们想要的物品，解锁他们的完整故事线。

## 核心玩法

- **经营店铺**: 管理库存、赚取金钱、提升声望
- **服务顾客**: 倾听顾客故事，满足他们的需求
- **收集物品**: 收集来自不同时代的稀有商品
- **解锁故事**: 完成顾客任务解锁完整的故事线

## 技术栈

- 引擎: Godot 4.2
- 语言: GDScript
- 平台: PC / 移动端

## 项目结构

```
time_shop_game/
├── project.godot          # Godot 项目配置
├── scenes/                # 游戏场景
│   ├── main_menu.tscn    # 主菜单场景
│   └── shop.tscn         # 商店主场景
├── scripts/               # 游戏脚本
│   ├── game_manager.gd   # 游戏管理器（全局单例）
│   ├── item_database.gd  # 物品数据库
│   ├── customer_database.gd  # 顾客数据库
│   ├── customer.gd       # 顾客类
│   ├── main_menu.gd      # 主菜单脚本
│   └── shop.gd           # 商店场景脚本
├── assets/                # 游戏资源
│   ├── sprites/          # 精灵图
│   ├── sounds/           # 音效
│   ├── fonts/            # 字体
│   └── data/             # 数据文件
└── addons/               # 插件

```
