# 时光杂货铺 - 完整增强版

## 🎮 游戏简介

《时光杂货铺》是一款温馨治愈的模拟经营+叙事解谜游戏。你将经营一家神奇的杂货铺，接待来自不同时代的顾客，倾听他们的故事，帮助他们找到珍贵的物品。

## 🎯 游戏玩法

### 基础流程
1. **开始游戏** - 选择新游戏或继续
2. **接待顾客** - 顾客会来访并讲述故事
3. **倾听对话** - 点击"继续"推进对话
4. **提供物品** - 根据需求从库存中选择物品
5. **获得奖励** - 完成交易获得金币和声望
6. **进货补充** - 通过神秘商人购买新物品
7. **商店升级** - 用金币升级商店获得更多优势
8. **收集故事** - 解锁所有顾客的故事

### 高级技巧
- **连击奖励** - 连续成功交易会增加声望奖励
- **声望解锁** - 高声望会吸引更多稀有顾客
- **合理进货** - 提前准备顾客需要的物品
- **升级优先级** - 先升级装修提升声望，再扩展仓库
- **成就追踪** - 关注成就系统获得额外满足感
